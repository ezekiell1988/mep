# Reference: Icon Simple Line Icons

## Uso básico

```html
<!-- Ícono Simple Line Icons -->
<i class="icon-{nombre}"></i>

<!-- Con tamaño personalizado -->
<i class="icon-home" style="font-size: 24px;"></i>
```

## Uso en botones

```html
<button class="btn btn-default">
  <i class="icon-home"></i> Home
</button>

<button class="btn btn-primary">
  <i class="icon-settings"></i> Configuración
</button>
```

## Uso en menús / listas

```html
<ul class="list-unstyled">
  <li><i class="icon-home fa-fw"></i> Inicio</li>
  <li><i class="icon-user fa-fw"></i> Perfil</li>
  <li><i class="icon-settings fa-fw"></i> Configuración</li>
</ul>
```
