# Reference: Icon Duotone (Iconify / Solar Duotone)

## Uso básico

```html
<!-- Ícono duotone via Iconify -->
<span class="iconify" data-icon="solar:{nombre}-bold-duotone"></span>

<!-- Con tamaño (Bootstrap display) -->
<span class="iconify display-6" data-icon="solar:home-bold-duotone"></span>
<span class="iconify display-4" data-icon="solar:home-bold-duotone"></span>
```

## Patrón de nombre: `solar:{nombre}-bold-duotone`

## Grid de visualización

```html
<div class="row gx-3">
  <div class="col-lg-3 col-md-4 col-6 pb-3 d-flex align-items-center fw-bold">
    <span class="iconify me-2 display-6" data-icon="solar:home-bold-duotone"></span>
    <span>solar:home-bold-duotone</span>
  </div>
</div>
```
