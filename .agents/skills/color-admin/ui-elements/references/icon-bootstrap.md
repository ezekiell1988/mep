# Reference: Icon Bootstrap (Bootstrap Icons v1.11.3)

## Uso básico

```html
<!-- Ícono básico -->
<i class="bi bi-{nombre-icono}"></i>

<!-- Con tamaño -->
<i class="bi bi-house fs-24px"></i>
<i class="bi bi-alarm fs-32px"></i>
```

## Grid de visualización de íconos (como en la página)

```html
<div class="row gx-3">
  <div class="col-lg-2 col-md-3 col-sm-4 col-6 text-center pb-3">
    <i class="bi bi-alarm fs-24px h-60px d-flex align-items-center justify-content-center bg-light mb-1"></i>
    alarm
  </div>
  <!-- repetir por cada ícono -->
</div>
```

## Convención de nombres

- Los íconos con variante rellena terminan en `-fill`
- Los íconos circulares terminan en `-circle` o `-circle-fill`
- Los íconos cuadrados terminan en `-square` o `-square-fill`
- Hay más de 2000 íconos disponibles en Bootstrap Icons v1.11.3
