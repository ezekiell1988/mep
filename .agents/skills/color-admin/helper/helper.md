# Skill: Color Admin — Helper CSS (Clases predefinidas)

## Propósito

Referencia completa de las utility classes propias de Color Admin que amplían Bootstrap 5.
Cubre desde spacing con valores en píxeles hasta paletas de colores con variantes 100–900.

**Disparar cuando:**
- Se necesite aplicar un color de fondo o texto con una clase (`.bg-blue-300`, `.text-red`)
- Se busque una clase de ancho/alto en píxeles (`.w-200px`, `.h-50px`)
- Se necesite margin/padding con valores en px extra (`.mt-10px`, `.p-15px`)
- Se quiera una clase de tipografía (`.fs-14px`, `.fw-600`, `.text-ellipsis`)
- Se pregunten las utility classes disponibles de Color Admin
